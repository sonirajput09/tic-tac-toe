# 🎮 Tic Tac Toe Game

An interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**.
Play against a simple AI opponent with a clean interface and score tracking.

---

## 🚀 Features

* 🎯 Player vs Computer gameplay
* 🤖 Basic AI (random move logic)
* 🔄 Restart game button
* 📊 Score tracking (X vs O)
* 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
